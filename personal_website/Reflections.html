Some occurances and issues while coding my website:

  • instead of inserting characters in Notepad++ it simply replaces the characters
      • - after some research I realized I was in 'Overwrite Mode' instead of 'Insert Mode'; in the bottom-right corner you can switch modes
  • trouble trying to align 2 parts of 1 line of text to opposing sides of the screen
      • - can use [style] tag to align text; use the [span] tag to format the 2nd part of the text to its respective alignment
  • when trying to insert an image, it only showed as a broken link
      • - i had to move the image to the same file folder so the tag can pull the image
